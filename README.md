# Turf Booking System

The Turf Booking System is a web-based application developed using the Django framework and MySQL database. The system allows users to book sports turfs online, turf owners to manage their turfs and bookings, and admins to manage the overall platform.

The project is designed with role-based access control and provides separate functionalities for Admins, Users, and Turf Owners.

# User Roles

The system supports three different user roles:

| Role | Description |
|------|-------------|
| Admin | Manages the entire platform |
| User | Books sports turfs |
| Turf Owner | Manages turf details and bookings |

---

# Features

## Admin
- Manage Users
- Manage Turf Owners
- Manage Turf Listings
- Monitor Bookings
- Control System Activities

## User
- Register and Login
- Browse Available Turfs
- View Turf Details
- Book Turf Slots
- View Booking History

## Turf Owner
- Add New Turf
- Update Turf Details
- Manage Turf Availability
- View User Bookings
- Accept or Reject Bookings

# System Workflow

1. Users register and log into the system.
2. Turf owners add and manage turf details.
3. Users browse available turfs and book slots.
4. Turf owners manage booking requests.
5. Admin monitors and manages all platform activities.
